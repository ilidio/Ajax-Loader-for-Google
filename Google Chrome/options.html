With this extension you can personalize Gmail, Google Calendar, Google Docs, Google Maps, Google Sites, Google Reader, Orkut and Google+ with an Animated Ajax Loader. This extension displays a loading image while requested content is being loaded by AJAX Requests.

Warning:

To use this extension, you might need (*) to load APNG javascript module to Google Chrome. To do that please install an APNG extension for Google Chrome available at the following link:
